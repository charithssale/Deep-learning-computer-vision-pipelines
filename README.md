# Deep Learning Computer Vision Pipelines

## Overview

This repository contains the final project for EE623/723 Computer Vision.

The project demonstrates the implementation of multiple deep learning based computer vision pipelines including image classification, object detection, vision transformers, image segmentation, and generative adversarial networks.

## Project Components

### 1. Deep Learning Image Classification

Implemented a convolutional neural network based image classification pipeline.

Tasks completed:

* Dataset preprocessing
* Model training
* Training loss analysis
* Validation loss analysis
* Classification performance evaluation

### 2. Object Detection

Implemented object detection using a pretrained deep learning model.

Tasks completed:

* Pretrained detector inference
* Detection on multiple object categories
* Bounding box visualization
* Confidence score analysis

### 3. Vision Transformer Classification

Implemented a Vision Transformer (ViT) based classification model.

Tasks completed:

* ViT inference
* Accuracy evaluation
* Comparison against CNN classifier

### 4. Image Segmentation

Implemented a segmentation pipeline using UNet.

Tasks completed:

* Dataset preprocessing
* Segmentation training
* Dice coefficient evaluation
* Segmentation mask generation

### 5. Generative Adversarial Networks

Implemented a GAN based image generation system.

Tasks completed:

* Generator training
* Discriminator training
* Synthetic image generation
* Generated image evaluation
